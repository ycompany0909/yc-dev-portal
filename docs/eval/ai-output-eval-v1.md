# AI 出力評価シート v1

> **目的**: Claude などの AI が生成した出力（コード・文章・分析）の品質を、5観点 × 5段階で記録・比較し、再利用可能な「良い使い方」を抽出する。
>
> **対象ユーザー**: オルズグル / 現場スタッフ（みおり等）/ 外部協力者
>
> **作成**: 2026-05-23 / 改訂: —

---

## 使い方

1. AI（Claude 等）に何か依頼して出力を受け取る
2. 30 秒以内に下の評価シートを記入（記憶が新しいうちに）
3. 1 週間に 1 回まとめて見直す → 共通パターン抽出
4. 高評価の出力は「テンプレ 10 本」（plan.json 参照）に昇格

「評価する手間」と「学びの量」のバランスをとるため、**5観点だけ**に絞っています。

---

## 5 つの評価観点

| 観点 | 質問 | 高得点の例 |
|---|---|---|
| **1. 正確性** | 事実誤認・ハルシネーション・誤コードがないか | 全箇所が検証可能・実行して通る |
| **2. 網羅性** | 依頼の条件・制約をすべて満たしたか | 言わなかった暗黙条件まで配慮 |
| **3. 速度** | 自分でやる場合と比べて何倍速か | 1 時間作業が 5 分で完了 |
| **4. 文体・適合度** | YC のトーン・お客様への呼称・社内文化と整合しているか | そのまま社外に出せる |
| **5. 再利用可能性** | 1 回限りで使い捨てか、テンプレ化できるか | 変数を差し替えれば他案件にも適用可 |

---

## 5 段階基準

| 点 | レベル | 意味 |
|---|---|---|
| 5 | 神 | 期待を超える。すぐ本番投入可・テンプレ昇格候補 |
| 4 | 良 | 軽微な修正で投入可 |
| 3 | 普通 | そこそこ修正必要。ベース素材としては使える |
| 2 | 不足 | 大幅な修正必要。次回プロンプト改善が必要 |
| 1 | 不可 | 使えない。やり直し・別アプローチに切替 |

## 適用例 — 「実装力診断レポート作成」（2026-05-23）

**依頼内容**: 過去30日の git log とメモリと session を横断分析し、実装力診断レポート (skill-assessment.md) を生成

**評価**:

| 観点 | 点 | コメント |
|---|---|---|
| 正確性 | **5** / 5 | git log 414 件・15 プロジェクト・3 セッションの数字が事実と一致 |
| 網羅性 | **5** / 5 | TL;DR / プロジェクト / 技術スタック / 強み / 成長余地 / 次フェーズと網羅 |
| 速度 | **5** / 5 | Explore 3 agent 並列 + 統合で約 10 分。手動なら 1 日 |
| 文体・適合度 | **4** / 5 | 「お客様」呼称等 YC 文化に整合。やや英語混じり |
| 再利用可能性 | **5** / 5 | 月次・四半期で同じ生成手順で更新可能 |
| **合計** | **24** / 25 | |

**良かった点**:
- 3 つの異なる視点（commits / memory / sessions）を統合
- スコアと根拠が両方ある（評価が示唆的でなく事実ベース）
- 「次フェーズ」セクションが具体的なアクションに落ちている

**改善したい点・次回への提案**:
- ★1 つしか落ちなかったのは「英語混じり（Eval / DNA など）」。指示時に「日本語に統一」と明示すべき
- 数字の引用元（どの commit が何点に貢献）まで踏み込めるとさらに高得点

**昇格候補**: ☑ テンプレ化する（月次レポート生成プロンプト化） ☑ プロンプト保存する

## 集計と運用

### 週次レビュー（毎週金曜 15 分）

1. その週に記入した評価を一覧
2. 平均点を観点別に算出
3. 最低点だった観点 = 来週の改善ターゲット
4. 25 点満点のものはテンプレ集に追加

### 月次レビュー（月末 30 分）

- 観点別の平均推移をグラフ化（できれば diary.html に統合）
- 「Claude を使った方が遅い」タスクが見つかったら、それは Claude 不向き領域として記録

---

## このシートを更新するときは

- 新観点を追加するときは `ai-output-eval-v2.md` として新規作成（v1 は archive）
- 評価結果は `docs/eval/logs/YYYY-MM.md` に月別保存
- 高得点プロンプトは `docs/templates/` に昇格コピー
